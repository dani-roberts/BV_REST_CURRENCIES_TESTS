## BV_REST_CURRENCIES_TESTS

E2E Test project for to test currency-pairs Api using RestAssured.


## Technologies
Project is created with:

- Java 11
- Apache Maven
- RestAssured
- TestNG


## Setup

- To build project

```
 mvn clean install -DskipTests
```

- Running the tests

```
 mvn clean test -Dtest=CurrencyPairs
```
